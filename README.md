## Run Locally

Clone the project

```bash
  git clone https://github.com/Ibcs16/game-dev-py
```

Go to the game directory

```bash
  cd my-project
```

Installing pygame

```python3 -m pip install -U pygame==2.6.0
```

Running a game

```bash
  python3 game/main.py
```
